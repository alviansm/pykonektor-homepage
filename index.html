Tentang

PYKonektor merupakan perangkat lunak untuk PC (Saat ini hanya tersedia untuk Windows) GUI untuk query database proyek dan BOM. Antarmuka (Graphical-user-interface) dan Fitur-fitur pengguna dibuat sesederhana mungkin agar alur penggunaan yang dilakukan bisa lebih cepat. Pengguna dapat mengoperasikan penambahan, update, hapus, dan pembacaan database (Versi saat ini support Database lokal menggunakan SQLite, yang kemudian dapat dimigrasikan ke MySQL).

Pengembangan program GUI dimulai dengan maksud agar operasi terhadap data-data proyek & BOM (Bill of Material) menjadi lebih mudah, pada saat melaksanakan program Magang Bersertifikat di PT INKA (Persero).
